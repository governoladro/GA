PITCH-VARIETY

[GENERIC-FUNCTION]
GA
Function Reference
ARGUMENTS:
  - pitches values [default = ()]  
  - ratio ideal-ratio [default = 0.7]

Description:

gives the ratio between the number of single values (i.e. values that do not repeat in the sequence)
and the total number of values in the sequence
<ratio> is the desired ratio (default 0.7)

Calcola il rapporto fra valori che non si ripetono e il totale dei valori
