PITCH-MUTATION

[GENERIC-FUNCTION]
GA
Function Reference
ARGUMENTS:
  - chromosomes values [default = ()]
  - prob mutation probability [default = 0.01]
  - mutation-type mutation-type [default = normal-add]

Description:

mutation operator for pitches. By a probability <prob> adds +/- 1 to the input values.
If <mutation-type> is set to 'fifth-add', adds +/- 7 (a fifth) or +/- 2 (a fifth of a fifth mod 12)

aggiunge +/-1 ai valori con probabilità definita in <prob>
Se <mutation-type> è impostato su 'fifth-add' aggiunge +/- 7 (una quinta)
oppure +/- 2, cioè la quinta della quinta ritrasopsta entro l'ottava
