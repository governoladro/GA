DISTANCES

[GENERIC-FUNCTION]
GA
Function Reference
ARGUMENTS:
  - chromosomes list of list of values [default = ()]  
  - allowed permitted elements [default = ()]  
  - type type [default = normal]
  - position consider the location [default = no]

Description:

judges each value based on its distance from the closest value in the list of <allowed>
scaling from 0 to 1, where 1 means best result.
the <type> value has three options:
normal (default)
octave (mod 12 of the values)
absolute (absolute value: useful for rhythmic values, since negative values are rests)
if <position> is 'yes' (default is 'no')the distance from the allowed value is multiplied by the position (index) the judged value has in the chromosome

giudica i valori in entrata in base alla distanza dal più vicino valore fra quelli presenti in <allowed>
riscala il risultato fra 0 e 1, dove 1 indica il risultato ottimale
Se il valore <type> (impostato solitamente su 'normal') è impostato su 'octave' calcola, il mod 12 degli elementi
se è impostato su 'absolute' calsola il valore assoluto.
utile ad esempio per i valori ritmici, dove le pause sono indicate con valori negativi
Se <position> è impostato su 'yes' moltiplica il valore calcolato per la posizione (indice) nel cromosoma del valore giudicato
