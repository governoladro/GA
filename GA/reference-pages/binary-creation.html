BINARY-CREATION

[GENERIC-FUNCTION]
GA
Function Reference
ARGUMENTS:
  - dimension how many chromosomes [default = 30]
  - chromlength chromosome length [default = 8]
  - max maximum lenght [default = 0]

Description:


Generates a list of binary chromosomes (the number of chroms generated is specified in <dimension>)
The length of each chromosomes is specified in chromLength.
If chromosomes of variable length are desired, the optional <;max> value indicates the maximum length, (and <chromLength> the minimum).
If <;max> is 0 or less than <chromLength> is not used
Output a list of lists of binary chromosomes.

Genera una lista di cromosomi binari (quanti è specificato in <dimension>) ognuno di lunghezza pari a <chromLength>.
Se si desiderano cromosomi di lunghezza veriabile, Il valore opzionale <max>
indica la lunghezza massima, (e <chromLength> quella minima). Non considerato se <max> è 0 o inferiore a <chromLength>
