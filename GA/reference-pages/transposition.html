TRANSPOSITION

[GENERIC-FUNCTION]
GA
Function Reference
ARGUMENTS:
  - chromosomes values [default = ()]
  - prob mutation probability [default = 0.05]
  - mutation-type mutation-type of mutation [default = fifth-add]
optional
  
  - section section [default = ()]    

Description:

transpose the musical phrase in input, with probability <prob>
adding +/- 1 to the input values if <mutation-type> is set to 'normal-add'
If <mutation-type> is set to 'fifth-add', adds +/- 7 (a fifth) or +/- 2 (a fifth of a fifth mod 12)
If the optional <section> is set to yes, the object makes the operation on a randomly chosen portion of the sequence

aggiunge +/-1 (se <mutation-type> è su 'normal-add') ai valori con probabilità definita in <prob>
Se <mutation-type> è impostato su 'fifth-add' aggiunge +/- 7 (una quinta)
oppure +/- 2, cioè la quinta della quinta ritrasopsta entro l'ottava
se il valore opzionale <section> è impostato su 'yes'
effettua la trasposizione solo su di una porzione della sequenza scelta casualmente
