ARGUMENTS:
  - chromosomes initial population [default = ()]

  - fitness-func fitness function [default = ()]

  - n-best-elements number of best elements [default = 1]
  - elite perc of unchange elements [default = 10]
  - scale rescaling method [default = prop]
  - result best of all/last generation [default = last-generation]

Description:

takes an initial population <chromosomes> and evolves it for a certain number of iterations <num-of-gen>,
judging its suitability <fitness-func> and carrying out mutation operations <mutation-func> on the chosen elements.
<elite> is the percentage of the best elements that pass to the next generation without undergoing mutations.
In <n-best-elements> it can be defined how many best elements the object returns at the end of the evolution process
If <result> is set to 'all-generations' the object returns <n-best-elements> from each generation,
otherwise only from the last.
the fitness rescaling method <scale> can be chosen between
proportional (each fitness is divided by the fitness average)
or sigma: (see documentation for detalis)
