PHRASE-GENERATOR

[GENERIC-FUNCTION]
GA
Function Reference
ARGUMENTS:
  - num-phrases how many phrases [default = 50]
  - low-note lowest note [default = -12]
  - high-note highest note [default = 12]
  - pitch-step cent division [default = 1]
  - min-duration dur1 [default = 1/16]
  - max-duration dur2 [default = 1/2]
  - dur-step dur def [default = 1/16]
  - total-duration total duration [default = 2]
  - perc-notes notes vs rests [default = 0.85]

Description:

Generate a <num-phrases> number of musical phrases of pitches between <low-note> and <high-note>,
rhythmic values between <min-duration> and <max-duration> (notes or rests in ratio or in milliseconds)
<pitch-step> is the minimum step between pitches (e.g. 100 midicents), while
<dur-step> is the minimal permitted rhythmic duration.
<total-duration> is the total duration of the phrase. If positive, indicates the exact duration of the chromosome (sum of notes and rests).
If negative, its absolute value indicates the maximum allowable duration of a chromosome.
<perc-notes> is the notes/rests ratio; 1 = only notes, 0 = only rests, 0.5 (default) eqaul distribution
Outputs a list of lists
