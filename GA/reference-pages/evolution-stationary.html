EVOLUTION-STATIONARY

[GENERIC-FUNCTION]
GA
Function Reference
ARGUMENTS:
  - chromosomes initial population [default = ()]
  - num-of-gen number of generations [default = 100]
  - fitness-func fitness function [default = ()]
  - mutation-func mutation function [default = ()]
  - n-best-elements number of best elements [default = 1]
  - elite perc of unchange elements [default = 10]
  - scale rescaling method [default = prop]
  - result best of all/last generation [default = last-generation]

Description:

In this type of selection only a small number of low fitness individuals are replaced,
leaving the others unchanged. The percentage of unchanged elements is defined in <elite>.
So we do not have a population entirely reconstructed from the descendants of the previous generation.
In <n-best-elements> it can be defined how many best elements the object returns at the end of the evolution process
If <result> is set to 'all-generations' the object returns <n-best-elements> from each generation,
and only from the last otherwise.

nell'evoluzione di tipo stazionario una percentuale definita di elementi <elite>
passano alla generazione successiva senza mutazione o incrocio
In questo modo non tutti gli individui della generazione seguente sono "evoluti" dagli individui della generazione precedente
Alcuni restano infatti inalterati
